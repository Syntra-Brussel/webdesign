Bij “desktop eerst” vertrekt de layout CSS van het grootste scherm (het desktop scherm) en worden aanpassingen (@media queries) voorzien voor de kleinere schermen (tablet, smartphone)

Bij “mobiel eerst” vertrek de layout CSS vanuit het kleinste scherm (de smartphone) en worden aanpassingen (@media queries) voorzien voor de grotere schermen (tablet en desktop scherm).

<blockquote>
Tot nu hebben we altijd layout CSS voorzien voor het desktop scherm. “Desktop eerst” dus.
</blockquote>
